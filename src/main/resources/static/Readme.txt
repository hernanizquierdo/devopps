Assessment 2 - DevOpps.
This software was developed using EclipseApp.


The compilation could be through:

From console terminal..
1. Install maven in the computer.	(Macintosh) : brew install maven
2. Find the directory. 				(Macintosh) : cd directory_code
3. Build the code. 					(Macintosh) : mvn clean install
4. Find the jar compilation 		(Macintosh) : cd target/java -cp devopps-0.0.1-SNAPSHOT.jar


From EclipseApp...
1. Download the branch.
2. Click right on the project. > Run As > Maven install
3. Click right on the project. > Run As > Maven clean
